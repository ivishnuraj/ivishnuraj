I'm a **Principal Software Architect** and **Engineering Leader** with over a decade of experience architecting high-availability systems, FinTech infrastructure, and enterprise-grade SaaS solutions. As the CEO of **[Synamatics Code Studio](https://synamatics.in)** , I lead cross-functional teams in building scalable platforms that bridge the gap between complex logic and seamless user experiences.

- **Optimized SAP-integrated order processing systems** to scale from **32,000 to 100,000 orders per hour**—a **212% throughput increase** achieved through architectural refactoring and asynchronous job queuing.

### ☁️ Cloud Infrastructure & DevOps
- **Architected Google Cloud environments** supporting **7+ concurrent applications** with **99.95% uptime**, leveraging **Kubernetes** for orchestration and fully automated CI/CD pipelines using Bitbucket Pipelines and Docker.

### 🌍 Logistics & E-Commerce Integration
- **Developed Mappy**, a custom API mapping middleware that unified global carriers (**FedEx, UPS, Australia Post, Aramex**) with major e-commerce platforms (**Shopify, eBay, WooCommerce, Magento**), reducing integration time by **70%** for client implementations.

### 💹 FinTech & Trading Systems
- **Specialized in MetaTrader 5 (MT5) server-side plugin development**, building custom indicators, automated commission engines, and high-frequency trading infrastructure for forex and CFD brokers.
